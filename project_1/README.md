### Описание проекта

Банк обратился с задачей разработать кампанию лояльности для удержания клиентов. Для этого нужно выяснить основные причины оттока.

---

### Какой кейс решаем

Необходимо определить, чем ушедшие клиенты отличаются от лояльных, и как связаны между собой различные признаки, влияющие на поведение клиентов.

**Условия задания:**
- В файле должно быть 10 графиков — 10 ответов к заданиям.
- Каждый график и преобразования к нему выполняются в отдельной ячейке.
- Под каждым графиком нужно дать ответ на вопрос и, если требуется, выводы.

**Метрика качества:** для каждого задания построен подходящий график, на его основе сделаны корректные выводы.

**Что практикуем:**
- написание кода на Python
- работа в IDE
- работа с GitHub
- визуализация данных
- формулировка выводов по графикам

---

### Краткая информация о данных

Данные представляют собой информацию о клиентах банка, включая демографические признаки, финансовые показатели и факт оттока.

---

### Этапы работы над проектом

1. Загрузка и первичный анализ данных
2. Очистка данных (при необходимости)
3. Построение 10 графиков по заданиям
4. Формулировка выводов к каждому графику
5. Оформление результатов

---

### Результат

Построены все требуемые графики, к каждому сформулированы выводы.

---

### Выводы

В ходе работы освоены базовые приёмы визуализации данных с помощью matplotlib и seaborn. Полученные графики позволили выявить ключевые различия между лояльными клиентами и теми, кто ушёл.
