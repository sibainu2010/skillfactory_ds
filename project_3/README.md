### Описание проекта

Кадровое агентство, подбирающее вакансии для IT-специалистов, ставит задачу: создать модель машинного обучения, которая будет рекомендовать вакансии клиентам, претендующим на позицию Data Scientist. Первый шаг — анализ данных (Data Understanding), чтобы понять, что из себя представляют данные и насколько они соответствуют целям проекта.

---

### Какой кейс решаем

Необходимо провести анализ данных и сделать выводы по каждому этапу.

**Условия задания:**
- Решение оформляется только в Jupyter Notebook.
- Решение оформляется в соответствии с ноутбуком-шаблоном.
- Каждое задание выполняется в отдельной ячейке.
- Текст SQL-запросов и код на Python должны быть читаемыми, с отступами.
- Выводы по каждому этапу оформляются в формате Markdown в отдельной ячейке.
- Выводы можно дополнительно проиллюстрировать графиками.
- Перед фиксацией работы в GitHub удалена ячейка с данными соединения.

**Метрика качества:** грамотный анализ всех данных и корректные выводы.

**Что практикуем:**
- написание кода на Python
- работа в IDE
- работа с GitHub
- анализ данных
- формулировка выводов на основе анализа

---

### Краткая информация о данных

Данные представляют собой информацию о вакансиях с платформы HeadHunter: требования к соискателям, описание обязанностей, требования к опыту, навыкам, образованию, а также информация о работодателях.

---

### Этапы работы над проектом

1. Знакомство с данными
2. Предварительный анализ данных
3. Детальный анализ вакансий
4. Анализ работодателей
5. Предметный анализ
