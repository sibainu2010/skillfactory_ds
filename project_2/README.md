### Описание проекта

Компания HeadHunter хочет построить модель для автоматического определения примерного уровня заработной платы, подходящей пользователю, на основе информации из резюме. Перед построением модели данные необходимо преобразовать, исследовать и очистить. Это и есть наша задача.

---

### Какой кейс решаем

Необходимо провести предобработку данных: преобразование, исследование и очистку.

**Условия задания:**
- Решение оформляется только в Jupyter Notebook.
- Решение оформляется в соответствии с ноутбуком-шаблоном.
- Каждое задание выполняется в отдельной ячейке.
- Код для каждого задания оформляется в одной-двух ячейках.
- Используется только пройденный материал: переменные, структуры данных, циклы, функции, библиотеки numpy, pandas, matplotlib, seaborn, plotly.
- Код должен быть читаемым, имена переменных и функций отражают их суть.
- Соблюдается руководство PEP 8.
- Графики содержат название и подписи осей.
- Выводы к графикам оформляются в формате Markdown под графиком в отдельной ячейке.

**Метрика качества:** грамотное выполнение всех операций над данными.

**Что практикуем:**
- написание кода на Python
- работа в IDE
- работа с GitHub
- визуализация данных
- формулировка выводов по графикам
- преобразование и очистка данных
- базовый анализ структуры данных
- разведывательный анализ данных

### Краткая информация о данных

Датасет содержит информацию из резюме, размещённых на платформе HeadHunter: опыт работы, образование, желаемая зарплата, город, сфера деятельности и другие признаки.
