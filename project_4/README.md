### Описание проекта

HR-агентство изучает тренды на IT-рынке труда. Компания хочет провести исследование на основе данных о зарплатах в сфере Data Science за 2020–2022 годы.

Задачи исследования:
1. Выяснить, какие факторы влияют на зарплату Data Scientist.
2. Ответить на ключевые вопросы:
   - наблюдается ли ежегодный рост зарплат у Data Scientist?
   - как соотносятся зарплаты Data Scientist и Data Engineer в 2022 году?
   - как соотносятся зарплаты Data Scientist в компаниях разных размеров?
   - есть ли связь между наличием должностей Data Scientist / Data Engineer и размером компании?

---

### Какой кейс решаем

Проводится разведывательный анализ данных (EDA) с последующими выводами, подкреплёнными визуализациями и статистическим тестированием.

**Условия задания:**
- Корректная загрузка данных
- Проверка на пропуски, дубликаты, типы данных
- Определение неинформативных признаков
- Классификация признаков на числовые и категориальные
- Расчёт основных статистических характеристик
- Базовый анализ каждого признака:
  - для числовых — гистограммы распределений
  - для категориальных — количество записей по категориям и визуализации
- Построение корректных визуализаций влияния признаков на зарплату
- Формулировка нулевых и альтернативных гипотез
- Выбор статистического теста с проверкой условий его применения (нормальность, равенство дисперсий)
- Проведение тестов:
  - для количественных признаков (одна выборка, две выборки, несколько выборок)
  - для категориальных признаков
- Структурированный ноутбук с заголовками и промежуточными выводами
- Визуализации с подписями осей и заголовками
- Итоговый вывод по исследованию
- Код оформлен по PEP-8 с комментариями

**Метрика качества:** грамотное проведение разведывательного анализа данных.

**Что практикуем:**
- написание кода на Python
- работа в IDE и GitHub
- анализ данных
- формулировка выводов
- статистическое тестирование
